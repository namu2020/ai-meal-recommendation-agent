===========================================
.env 파일 생성 가이드
===========================================

프로젝트 루트 디렉토리 (crewai-food-app/)에 
'.env' 라는 이름의 파일을 생성하고 
아래 내용을 복사하여 붙여넣으세요.

------- 아래 내용 복사 -------

OPENAI_API_KEY=your_openai_api_key_here
OPENAI_MODEL=gpt-4o-mini

------- 여기까지 복사 -------

⚠️ 중요:
1. 'your_openai_api_key_here' 부분을 
   실제 OpenAI API 키로 바꿔주세요!
   
2. API 키는 https://platform.openai.com/ 에서 발급받을 수 있습니다.

3. .env 파일은 절대 GitHub에 커밋하지 마세요!
